# Add Claude Rule

This command helps add new rules or guidelines to the CLAUDE.md file.

## Usage
```
/add-claude-rule
```

## Description
Adds a new rule, guideline, or best practice to the CLAUDE.md file. This command will:
1. Read the current CLAUDE.md file
2. Prompt for the rule category (e.g., "Development Guidelines", "Architecture Rules", "Code Standards")
3. Prompt for the rule description
4. Add the rule to the appropriate section in CLAUDE.md
5. Create new sections if needed

## Example
After running the command, you might add rules like:
- Code formatting standards
- Testing requirements
- Architecture constraints
- Security guidelines
- Performance considerations

## Notes
- Rules are added in a structured format
- Existing sections are preserved
- New sections are created as needed
- Changes are made directly to CLAUDE.md
