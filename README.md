# PowerPoint Slide Creation Automation

PowerPoint slides generation automation project using Python and python-pptx library.

## Features

- Automated PowerPoint slide generation
- SuccessFactors data integration
- Professional slide templates
- Batch processing capabilities

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the main script:

```bash
python successfactors_professional.py
```

## Files Structure

- `successfactors_professional.py` - Main professional slide generator
- `successfactors_ppt_improved.py` - Improved version with enhanced features
- `successfactors_ppt_generator.py` - Basic slide generator
- `requirements.txt` - Python dependencies
- `01_分析レポート/` - Analysis reports and generated presentations
- `02_傾向分析/` - Trend analysis documentation

## Requirements

- Python 3.8+
- python-pptx
- openpyxl
- pandas
